# UI/UX Design

## Web App
- **Patient panel**: book appointments, view history, receive notifications.
- **Professional panel**: manage availability, view appointments, sync with Google Calendar.
- **Admin panel**: user management, service configuration, exception handling.

## Mobile App
- **Patient-focused**: booking, push notifications, profile management.
- **Professional-focused**: calendar view, appointment list, push notifications for upcoming appointments.

## Design Principles
- Accessibility first (WCAG 2.1 AA).
- Responsive design for all screen sizes.
- Clear, simple language for users with limited technical experience.
